启动脚本

国外机器

wget --no-check-certificate -O gost.sh https://raw.githubusercontent.com/lusalem/EZgost/refs/heads/main/gost.sh && chmod +x gost.sh && ./gost.sh

国内机器

wget --no-check-certificate -O gost.sh https://raw.githubusercontent.com/lusalem/EZgost/refs/heads/main/gost.sh && chmod +x gost.sh && ./gost.sh

再次运行本脚本只需要输入./gost.sh回车即可
